# BevelDeformer — план разработки

Этот файл — рабочий план (не changelog).

## Сделано (v0.1.x)

- Пакет аддона в [addon/bevel_deformer](addon/bevel_deformer) (модули + единая панель)
- Логотип в Preferences (через previews)
- Выбор `World Axis` (X/Y/Z) для определения locked направления при создании lattice
- Более безопасная регистрация/cleanup при ошибках и read-only состоянии
