# Updates Log

Changelog для аддона. Этот файл ведём как “короткий список релизов”, без старой истории.

## 2026-01-12 — v0.1.6 (Blender 4.5.0)

- UI/UX:
  - Вкладка панели перенесена в `Bevel_Deform`
  - Добавлен `Apply Lattice` (apply модификатора + удаление lattice, если больше не используется)
  - `Delete Lattice` и `Apply Lattice` выровнены в одну линию
- Lattice:
  - `Locked Axis` (bool): при включении locked-ось всегда имеет разрешение 2
  - При выключенном `Locked Axis` адаптивное разрешение применяется по всем осям
  - Locked-метаданные сохраняются внутри каждого созданного lattice (Custom Properties), чтобы переключение сценовых настроек не ломало существующие lattice
- Deform:
  - Live Preview стал стабильнее (таймер переживает сбои и корректно перерегистрируется)
  - Shift Factor: диапазон -1..1, дефолт 0
  - Scale Factor: дефолт 1
  - Добавлены Offsets (X/Y/Z) для управляемой «длины/ширины/высоты» lattice
  - Reset Selected Lattices дополнительно сбрасывает ползунки (Scale=1, Shift=0, Offsets=0)
- Выделение:
  - Деформация и reset поддерживают смешанное выделение (меши и/или lattice)

## 2026-01-11 — v0.1.0

- Собран полноценный аддон-пакет в [addon/bevel_deformer](addon/bevel_deformer) (модульная структура вместо одного файла)
- Добавлен логотип для отображения в настройках аддона (Preferences → Add-ons)
- Единая панель управления: View3D → Sidebar → вкладка `Bevel`
- Lattice:
  - Создание lattice per-mesh + подтверждение перезаписи существующих lattice
  - Удаление lattice для выбранных мешей/лэттисов
  - Выбор `World Axis` (X/Y/Z) для направления, по которому определяется locked ось
- Deform:
  - Сброс lattice в равномерную сетку
  - Shift + условный scale по тем осям, где был shift
- Стабильность:
  - Защита от read-only состояния (через обработку RuntimeError) без крашей UI
  - Убран `select_all()` из создания lattice, чтобы не ломать выделение/контекст
- Старые скрипты перенесены в [Legacy](Legacy) и не участвуют в установке аддона
